==============
Reporting Bugs
==============


If you encounter any problem when using ``smcsmc``, please file a short
bug report by using the `issue tracker <https://github.com/luntergroup/smcsmc/issues>`_
on GitHub or email joe.zhu (at) well.ox.ac.uk.

Please include the output of ``smcsmc -v`` and the platform you are using
``smcsmc`` on in the report. If the problem occurs while executing ``smcsmc``,
please also include the command you are using and the random seed.

Thank you!
